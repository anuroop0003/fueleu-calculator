SFOC CORRECTION CALCULATOR  ·  NAVCIIFUEL
User Guide
This calculator implements the ISO 3046-1 standard for correcting Specific Fuel Oil Consumption (SFOC) measurements from actual site conditions to standard reference conditions. It supports up to 3 engines simultaneously, computes ISO correction factors (κ, α, β), applies calorific value correction, and produces fleet-averaged results with visual charts — suitable for sea trials, shop tests, and performance benchmarking.
01 Step-by-Step Workflow
1
Set ISO 3046 Reference Conditions
At the very top of the page is the ISO 3046 Reference strip. These four values define the standard atmosphere against which all corrections are made. Defaults follow ISO 3046-1 exactly. Change only if your engine certificate specifies non-standard reference conditions.
pr = 1000 hPa Tr = 298 K (25°C) fir = 0.30 Tcr = 298 K
2
Add Engines & Enter Engine ID
Click + Add Engine to create up to 3 engine tabs. Each tab has its own independent data table. Fill in Engine No., MCR [kW] (critical — used to compute generator output D and 25% load SFOC), and an optional Order No. for identification. Switch between engines using the tabs. Remove an engine by clicking the ✕ on its active tab.
Up to 3 engines MCR required
3
Update Load Columns
Use + Add Load to add a new engine load point, or click × on an existing load chip to remove it. The table columns, fleet-average summary, and SFOC comparison charts update automatically to match the selected load points.
+ Add Load × Remove Load Columns · Summary · Charts
4
Enter Measurement Inputs (White rows)
White rows are direct measurement inputs from the test. Enter values for the active load columns shown in the table: A — fuel mass measured (kg), C — measurement duration (s), Tx — ambient air temperature at TC inlet (°C), Tcx — charge air coolant temperature (°C). These are the raw field observations.
A · C · Tx · Tcx From test certificate
5
Verify Default Values (Green rows)
Green rows are engine-specific constants pre-filled from typical ISO 3046-1 values. Review and override with values from your engine maker's test report: B (buoyancy factor), Px (ambient pressure, hPa), G (measured NCV, MJ/kg), H (standard NCV, MJ/kg), ηm (mechanical efficiency), exponents m, n, s, and generator efficiency I (ηg).
Editable defaults Engine maker constants
6
Read Calculated Outputs (Blue & Yellow rows)
Blue rows show intermediate ISO correction steps computed automatically: generator output D, measured SFOC, correction factor k, alpha α, beta β, and LCV-corrected SFOC. Yellow rows display the final SFOC ISO Corrected result (g/kWh) — the value for official reporting. No manual entry is needed in these rows.
Blue = intermediate steps Yellow = ISO result
7
Calculate
Click Calculate after entering the required engine data. If any required field is empty, the field is highlighted in red so it can be completed. Once the required inputs are valid, the calculator shows the ISO corrected results, fleet-average summary, and SFOC comparison charts.
Calculate Red = required field Results · Summary · Charts
8
Reset Defaults
Use Reset defaults at the bottom of any engine panel to restore only that engine's green-row constants. This does not affect other engines or the white measurement input fields.
Per-engine reset Green-row constants only

03 Calculation Formulas (ISO 3046-1)
SFOC Measured
SFOCm = (B × A × 1000 / C) × 3600 / (load% × MCR)
A = fuel mass (kg)  ·  B = buoyancy factor  ·  C = time (s)  ·  MCR = rated power (kW)
Ambient Correction Factor k
k = (Px / pr)m × (Tr / (Tx+273.15))n × (Tcr / (Tcx+273.15))s
Px, Tx, Tcx = site conditions  ·  pr, Tr, Tcr = ISO reference values
Alpha (α) — Mechanical Efficiency Correction
α = k − 0.7 × (1 − k) × (1/ηm − 1)
Beta (β)
β = k / α
SFOC LCV Corrected
SFOCLCV = SFOCm × G / H
G = actual fuel NCV  ·  H = standard reference NCV
SFOC ISO Corrected — Final Result
SFOCISO = (SFOCLCV / β) × ηg
ηg = generator efficiency (I)

05 Important Notes
Accuracy: Results are only as accurate as the input data. Ensure all measured values (A, C, Tx, Tcx, Px) come from calibrated instruments at stable, steady-state load conditions during the test.
Temperature Units: Tx and Tcx must be entered in °C — the calculator automatically converts to Kelvin internally using +273.15. The ISO reference temperatures Tr and Tcr in the top strip are in Kelvin.
Fleet Average: When multiple engines are active, the Summary section displays an arithmetic mean of ISO-corrected SFOC across all engines at each load point. This is the standard method for fleet performance reporting.
Download: The downloaded HTML file is fully self-contained — it includes Chart.js inline and preserves all entered data. It can be opened in any browser offline, emailed, or printed as a record.
