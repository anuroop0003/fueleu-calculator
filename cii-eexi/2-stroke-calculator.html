User Guide
2 Stroke Engine ISO SFOC Correction Calculator  ·  ISO 3046-1
This calculator is designed for 2 Stroke Engine SFOC correction using the ISO 3046-1 reference condition approach. It converts measured SFOC and actual site conditions into ISO-corrected SFOC at 25%, 50%, 75% and 100% MCR load points. The tool also provides a dedicated SFC Calculator for PME, allowing the ISO-corrected SFC at the required propulsion power point to be determined directly from the corrected SFOC curve.
01 Step-by-Step Workflow
1
Set ISO 3046 Reference Conditions
Use the ISO 3046 Reference strip at the top of the page to confirm the reference values used for correction. The default values are Tr = 25°C, pr = 1000 mbar, Tcr = 25°C and H ref LCV = 42700 kJ/kg. Change these values only when the engine certificate or technical file specifies different reference conditions.
Tr = 25°C pr = 1000 mbar Tcr = 25°C LCV ref = 42700 kJ/kg
2
Enter Engine Identification
In the 2 Stroke Engine panel, enter the engine number and optional order number for traceability. The tool is configured as a single 2 stroke engine calculation sheet, so all input, correction and result rows relate to the same main engine.
Engine No. Order No. optional
3
Enter Measured SFOC Values
In the SFOC Measured row, enter the measured SFOC values from the shop test, sea trial or test log for the four load points: 25%, 50%, 75% and 100% MCR. These values are entered directly in g/kWh.
SFOC measured g/kWh
4
Enter Actual Ambient and Fuel Conditions
Enter the actual site condition values for each load point: Tx for ambient air temperature, Px for barometric pressure, Tcx for charge air coolant temperature and G for fuel lower calorific value. Green default rows can be adjusted when actual test data is available.
Tx °C Px mbar Tcx °C G kJ/kg
5
Review Automatic ISO Correction Steps
The calculator automatically determines the correction contribution from ambient temperature, barometric pressure, coolant temperature and fuel LCV. These intermediate rows show ΔT correction, ΔP correction, ΔTcx correction, ΔLCV correction and the total Σ corrections.
Blue = calculated Read only
6
Review SFOC ISO Corrected — Results
The SFOC ISO Corrected — Results section displays the final corrected SFOC in g/kWh for each load point. The result cards show the corrected values at 25%, 50%, 75% and 100% MCR, making it easy to review, verify and use the values for reporting.
Yellow = final result ISO corrected
Result section: Use SFOC ISO Corrected — Results as the main output area for the final corrected values before reviewing the graph section.
7
Use the SFC Calculator for PME
The SFC Calculator for PME uses the ISO-corrected SFC reference curve from the engine table. If MCRlim is available, enter PME and MCR to calculate the load percentage and interpolate SFC. If MCRlim is not available, the tool uses the fixed 75% MCR ISO-corrected SFC value.
MCRlim Yes = interpolation MCRlim No = 75% MCR
8
Calculate, Reset Defaults and Load Points
Use the Calculate button to refresh the correction table, result cards and charts after entering or editing values. Use Reset defaults to restore the standard default values. Use + Add Load or × to update columns, summary and charts.
Calculate Reset defaults + Add Load / ×

03 SFC Calculator for PME — Detailed Guidance
The ⚙️ SFC Calculator for PME determines the ISO-corrected specific fuel consumption at the required propulsion power point. It uses the SFOC ISO Corrected values generated from the main engine table as the reference curve, then either interpolates the SFC value at the calculated PME load point or uses the fixed 75% MCR value depending on the MCRlim selection.
A
ISO Corrected SFC Reference Curve
The reference table inside this section is automatically populated from the engine calculation table above. It shows the ISO-corrected SFC values at 25%, 50%, 75% and 100% MCR. These values form the curve used for the PME calculation.
From engine table ISO corrected curve g/kWh
B
MCRlim Available — Yes
Select Yes when MCRlim / applicable MCR value is available. Enter PME in kW and MCR in kW. The calculator determines the operating load percentage using Load% = PME / MCR × 100, then calculates SFCPME by linear interpolation between the nearest ISO-corrected SFC load points.
PME input required MCR input required Linear interpolation
C
MCRlim Available — No
Select No when MCRlim is not available. In this mode, no PME or MCR input is required. The calculator directly uses the 75% MCR ISO-corrected SFC value from the reference curve as SFCPME.
Fixed 75% MCR No interpolation
D
Result Display and Graph Marker
After calculation, the section displays SFCPME — ISO Corrected in g/kWh, the selected or calculated load point, and the method used. The SFCPME point is also plotted on the chart for visual checking against the ISO-corrected SFC curve.
SFCPME result Graph point Method shown
PME SFC Calculation Logic
If MCRlim = Yes: Load% = PME / MCR × 100 → SFCPME by interpolation
If MCRlim = No: SFCPME = ISO Corrected SFC at 75% MCR
The PME result is linked to the corrected engine table, so any change in measured SFOC or correction inputs automatically updates the PME reference curve and result.
04 Calculation Formulas
Ambient Temperature Correction
ΔT = (Tx − Tr) / 10 × 0.2%
0.2% correction per 10°C difference from ISO reference ambient temperature.
Barometric Pressure Correction
ΔP = −(Px − pr) / 10 × 0.02%
Pressure correction is applied relative to the ISO reference pressure.
Coolant Temperature Correction
ΔTcx = (Tcx − Tcr) / 10 × 0.6%
0.6% correction per 10°C difference from ISO reference coolant temperature.
Fuel LCV Correction
ΔLCV = −(G − Href) × 100 / Href
Corrects measured SFOC for the difference between actual fuel LCV and reference LCV.
Total Correction
Σ corrections = ΔT + ΔP + ΔTcx + ΔLCV
SFOC ISO Corrected
SFOCISO = SFOCmeasured − (Σ corrections × SFOCmeasured / 100)
PME Load Percentage
Load% = PME / MCR × 100
Used when MCRlim is available; SFCPME is then interpolated from the ISO-corrected SFC curve.
05 Row Color Coding
Table Row Color Reference
White — Direct measurement inputs entered by the user.
Green — Editable default values that may be changed.
Blue — Intermediate correction calculations generated automatically.
Yellow — Final ISO-corrected SFOC result rows.
06 Important Notes
Input quality: The correction result depends directly on the measured SFOC, ambient condition and fuel LCV values entered in the table. Use values from calibrated test records wherever possible.
Units: Enter temperature values in °C, pressure in mbar, fuel LCV in kJ/kg and SFOC in g/kWh.
SFC for PME: When MCRlim is selected as Yes, the tool uses interpolation between the ISO-corrected SFOC load points. When selected as No, the 75% MCR ISO-corrected value is used directly.
Results and graph check: First review SFOC ISO Corrected — Results, then use the graph to compare measured SFOC, ISO-corrected SFOC and the SFCPME point for visual verification before reporting.
