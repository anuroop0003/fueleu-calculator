CII Calculator — Page Guide
COMPLETE FUNCTION & COLUMN REFERENCE
This calculator computes the Carbon Intensity Indicator (CII) — specifically the Annual Efficiency Ratio (AER) — for vessels as required under MEPC.354(78) / MARPOL Annex VI. Complete sections A and B in order, then click Calculate Now to view the rated result in section C.
Step-by-Step Workflow
1
Select Ship Type & Enter Vessel Particulars
Open Section A. Choose your vessel's Ship Type first — this sets the correct capacity unit (DWT or GT) and the MEPC coefficients for Required CII. Then fill in IMO, Vessel Name, Year, Capacity, and Distance (nm). Set the Reporting Period dates — the Year field auto-fills from these dates.
2
Enter Fuel Consumption in Section B
Open Section B. For each fuel type consumed during the year, select the fuel from the drop-down — the CF value auto-fills. Enter the total Consumption (tonnes) from your Bunker Delivery Notes. Use + Add Row for multiple fuels. Alternatively, click Excel Template to download a spreadsheet, fill it in, then Upload Excel to import all rows at once.
3
(Optional) Configure Correction Factors
Still in Section B, scroll down to the Correction Factors toggle. If the vessel has voyage adjustments, transferred fuel, electrical/boiler fuel deductions, or EEXI-related corrections, switch to Yes. Set each factor to Applicable and click Modify to enter its values. Correction factors adjust both CO₂ and distance for the official corrected CII figure.
4
Click "Calculate Now"
Press Calculate Now in the top toolbar. Section C will appear with: Uncorrected and Corrected CO₂ & Distance, Attained CII (uncorrected and corrected), Required CII, and the A–E Rating. The compliance chart and year-on-year trend table also populate instantly.
5
Review the Results & Rating in Section C
Review the KPI cards, the A–E rating band, and the chart. The coloured band indicator shows where the vessel sits relative to the Required CII boundaries. The Year Trend Table on the right shows how the rating will change through 2030 if the same operational profile continues.
6
Save, Export or Print
Use Save XML to archive all inputs for future re-import. Use Print / Save PDF to generate a formal compliance report. To share with a colleague, save the XML and send it — they can use Import XML to restore the full session exactly.
Top Toolbar Buttons
Calculate Now Runs all CII calculations using the current Part A and Part B inputs. Section C appears with results, rating, and the compliance chart. Re-click after any change to refresh.
Save XML Exports all current input values as a structured XML file. Use to archive a session or share with colleagues. The file captures vessel data, all fuel rows, and correction factor settings.
Import XML Restores all input values from a previously saved XML file. All fields (including fuel rows and correction factors) are repopulated exactly as they were when the file was saved.
Print / Save PDF Generates a formatted A4 compliance report with all inputs, fuel breakdown, and CII results. Use your browser's Print → Save as PDF to save. Ensure calculation has been run first.
Reset Clears all input fields, removes any uploaded Excel data, hides Part C results, and resets fuel rows to a single blank entry. Use to start a completely fresh calculation.
Section A — Vessel & Year Inputs
Enter the vessel's core identifying data and the annual reporting period. All fields affect the CII calculation directly.
Vessel Name Free-text identifier for the vessel. Appears on the printed PDF report. Does not affect calculations.
IMO Number The vessel's unique 7-digit IMO identification number. Appears on the report only — does not affect calculations.
Ship Type Select the vessel's MARPOL Annex VI ship type category. This determines the a and c coefficients used to compute Required CII, and whether the capacity is measured in DWT or GT. Types include bulk carriers, tankers, container ships, gas carriers, and more.
Year The reporting year (2023–2030). Each year has a progressive required CII reduction factor under MEPC.354(78). Selecting a later year applies a stricter reduction target.
Capacity (DWT or GT) The vessel's deadweight tonnage (DWT) or gross tonnage (GT), depending on ship type. This is the denominator in the AER formula: AER = CO₂ (g) ÷ (Capacity × Distance).
Metric Fixed at AER (Annual Efficiency Ratio) for this calculator, per MEPC guidelines. Displays as read-only information.
Distance (nm) Total distance sailed during the reporting year in nautical miles. Source from the vessel's official voyage records or logbook. Used as the denominator in the AER formula alongside capacity.
Reporting Period (From / To) The start and end dates of the annual reporting period (typically 1 Jan – 31 Dec). Appears on the printed report and is used to auto-fill the Year field. Both dates must fall within the same calendar year.
Section B — Fuel Consumption
Enter each fuel type consumed by the vessel during the reporting year. Multiple rows can be added. CO₂ is computed as: Consumption (t) × CF (tCO₂/tFuel).
Fuel (column) Select a fuel type from the drop-down. Options include HFO, LFO, MDO, MGO, LNG, methanol, ethanol, LPG, and more. For non-listed fuels, select Other and enter the name manually.
CF (tCO₂/tFuel) The carbon conversion factor for the selected fuel, per MEPC.354(78) Table 2. This field is auto-populated when a standard fuel is selected. For custom fuels, enter the value manually. CF represents how many tonnes of CO₂ are produced per tonne of fuel burned.
Consumption (t) Total mass of this fuel consumed during the reporting year in metric tonnes. Source from the vessel's Bunker Delivery Notes (BDNs) and fuel records. This is the primary driver of CO₂ emissions.
Add Row / Remove Row Use + Add Row at the bottom to include additional fuel types. Use the Remove button on each row to delete it. There is no limit to the number of fuel rows.
Excel Template Downloads a pre-formatted .xlsx file with the correct column headers (Fuel, CF, Consumption). Fill in the consumption column for each fuel, then upload the completed file.
Upload Excel Imports the completed Excel (.xlsx, .xls) or CSV file and populates all fuel rows in the table automatically. Matches fuel names to the built-in list and applies the correct CF values. Unrecognised fuels are set to Other.
Clear All Removes all current fuel rows and replaces them with a single blank row. Use after uploading incorrect data or to start the fuel section fresh.
Correction Factors Toggle to Yes to display the Correction Factors table (FCvoyage, TF, FCelectrical, FCboiler, FCothers, EEXI). Each factor can be set as applicable and configured via the Modify button. These adjust both CO₂ and distance for corrected CII output per MEPC guidelines.
Section C — Results
Appears after clicking Calculate Now. No inputs required — all values are calculated automatically.
Uncorr. CO₂ Emitted (t) Total CO₂ calculated directly from fuel consumption × CF, before any correction factors are applied. Formula: Σ (Fueli × CF,i).
Corr. CO₂ Emitted (t) CO₂ after applying all active correction factors (FCvoyage, TF, FCelectrical, FCboiler, FCothers). If no correction factors are set, this equals Uncorrected CO₂.
Uncorr. Distance (NM) The raw distance entered in Section A, before any distance correction factor is applied.
Corr. Distance (NM) Distance after applying the FCvoyage correction factor (if applicable). This is used as the denominator in the corrected AER calculation.
Uncorr. CII (Attained) The raw Attained CII using uncorrected CO₂ and uncorrected distance. Formula: AER = CO₂ (g) ÷ (Capacity (t) × Distance (nm)). Units: gCO₂/(t·nm).
Corr. Attained CII The corrected Attained CII using corrected CO₂ and corrected distance. This is the official figure used for rating comparison. Formula: AERcorr = CorrCO₂ (g) ÷ (Capacity × CorrDistance).
Required CII The Required CII for the vessel in the selected year. Calculated as: Required CII = (a × Capacity−c) × (1 − Zyear/100), where a and c are ship-type coefficients from MEPC.354(78) Table 1, and Z is the year-specific reduction factor.
Rating (A–E) The vessel's CII rating band, determined by comparing Attained CII to the Required CII boundary multipliers d1–d4: A = Superior (Attained < d₁ × Required) · B = Above average · C = Average · D = Below average · E = Inferior (Attained > d₄ × Required).
CII Trend Chart Displays the vessel's Attained CII plotted alongside Required CII boundary lines for A/B/C/D/E bands across the years 2023–2030. The selected year is highlighted. Hover over the chart to read exact values.
Year Trend Table Shows the Required CII, Attained CII, and calculated rating for every year from 2023 to 2030, based on current inputs. Useful for planning future compliance trajectory.
